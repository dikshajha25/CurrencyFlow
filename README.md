# 🌍 CurrencyFlow

A modern and responsive **Currency Converter** built using **React + Vite**. It allows users to convert currencies in real-time using live exchange rates from a public currency API. The application features a clean glassmorphism UI, responsive design, and a smooth user experience.

---

## 🚀 Live Demo

https://currencyflow.vercel.app

---

## ✨ Features

- 🌍 Real-time currency conversion
- 💱 Live exchange rates
- 🔄 Swap currencies with one click
- 📱 Fully responsive design
- 🎨 Modern Glassmorphism UI
- ⚡ Built with React + Vite
- 🛡️ Error handling
- ⏳ Loading state while fetching data
- 📜 Recent conversion history (Local Storage)

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CurrencyFlow.git
```

Move into the project folder

```bash
cd CurrencyFlow
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
