## вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CurrencyFlow.git
```

Move into the project folder

```bash
cd CurrencyFlow
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
